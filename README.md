# AI Photofun Studio

A comprehensive AI-powered photo editing and manipulation studio application developed as part of the Introduction to Software Engineering course (CSC13002) by Group 7, Class 23CNTN, VNU-HCMUS.

## Overview

AI Photofun Studio is an advanced web application that leverages artificial intelligence to provide a wide range of photo editing and generation capabilities. The platform offers multiple AI-powered tools for creative photo manipulation, enhancement, and generation.
